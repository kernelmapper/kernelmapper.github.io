    Link: https://doi.org/10.1145/2535930 What is CORFU? CORFU is distributed shared log to support strong consistency, fault tolerance, durability and total order. It can be building block of various distributed systems like: replicated transactional systems (e.g., Hyder) consensus engine replicated state machines metadata services Design Client CORFU client has very simple interfaces - append, read, trim and fill. In CORFU, each position in the shared log is projected onto a set of storage pages on different storage units.
