Scalog: Seamless Reconfiguration and Total Order in a Scalable Shared Log

Posted at — Aug 22, 2020

Link: https://www.usenix.org/conference/nsdi20/presentation/ding

Scalog

Scalog is another shared log similar to CORFU. Like CORFU, it has Data Layer that stores logs. Unlike CORFU, it has no sequencer, however, it has Ordering Layer.

The novel approach of the Scalog is the ordering layer and its append workflow.

Ordering Layer

The ordering layer periodically summarizes the fully replicated prefix of the primary log segment of each storage server in a cut, which it then shares with all storage servers. … The storage servers use these cuts to deterministically assign a unique global sequence number to each durable record in their log segments
